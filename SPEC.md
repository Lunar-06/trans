# 文旅电商平台 - 项目规范

## 1. 概念与愿景

这是一个融合文化体验与电商功能的文旅平台，旨在推广地方文化特色，让用户在购物的同时感受当地民俗风情。整体风格呈现"传统与现代交织"的氛围，既有文化底蕴又不失便捷的购物体验。

## 2. 设计语言

### 美学方向
- 新中式风格：融合传统水墨元素与现代简洁设计
- 以中国传统文化为视觉基础，搭配现代UI组件

### 色彩方案
- 主色：#8B4513（赭石棕）- 代表大地与传统文化
- 次色：#F5F5DC（米色）- 温暖的背景色调
- 强调色：#CD853F（秘鲁色）- 用于按钮和交互元素
- 辅助色：#2F4F4F（墨绿）- 文字与标题
- 背景色：#FFFAF0（花白色）- 页面底色

### 字体
- 标题：思源宋体 / Noto Serif SC
- 正文：思源黑体 / Noto Sans SC
- 英文/数字：Poppins

### 空间系统
- 基础间距单位：8px
- 常用间距：16px, 24px, 32px, 48px
- 卡片圆角：12px
- 阴影：0 4px 12px rgba(0,0,0,0.08)

### 动效哲学
- 页面切换：淡入淡出，300ms ease
- 卡片悬停：轻微上浮 + 阴影加深，200ms
- 按钮点击：缩放至0.98，150ms
- 列表加载：从下往上滑入，stagger 100ms

- 桌面端：最大宽度1200px，多列布局
- 平板端：768px-1200px，两列布局
- 移动端：<768px，单列布局，底部导航

## 4. 功能与交互

### 4.1 AI方言翻译
- 输入框：支持文本输入，提供方言选择下拉菜单
- 方言选择：普通话、四川话、广东话、闽南语、吴语等
- 翻译按钮：点击后调用后端API（预留接口）
- 结果展示：翻译结果以卡片形式展示，支持复制功能
- 状态：加载中显示loading动画，错误时显示友好提示

### 4.2 购物平台
- 商品列表：网格布局展示商品卡片
- 商品卡片：图片、名称、价格、加入购物车按钮
- 分类筛选：顶部标签栏支持分类切换
- 购物车：侧边栏展示已选商品，可修改数量
- 结算流程：确认订单 -> 填写信息 -> 提交订单

### 4.3 民俗故事介绍
- 故事卡片：封面图 + 标题 + 简介
- 详情页：点击卡片进入故事详情
- 内容展示：富文本展示民俗故事、历史传说等
- 分类标签：地区、类型（传说、习俗、节庆等）

## 5. 组件清单

### 导航栏 (NavBar)
- 默认：透明背景，深色文字
- 滚动后：白色背景，阴影显现
- 当前页面：高亮显示

### 按钮 (Button)
- 主要按钮：棕色调，悬停加深
- 次要按钮：边框样式
- 禁用状态：灰色，不可点击
- 加载状态：显示loading spinner

### 输入框 (Input)
- 默认：底部边框
- 聚焦：边框颜色变为强调色
- 错误：边框变红，显示错误信息

### 卡片 (Card)
- 默认：白色背景，轻微阴影
- 悬停：上浮4px，阴影加深
- 商品卡片：图片比例4:3

### 模态框 (Modal)
- 居中显示，背景遮罩
- 关闭按钮在右上角
- 动画：淡入 + 轻微缩放

## 6. 技术方案

### 前端技术栈
- 框架：Vue 3 + Vite
- 路由：Vue Router 4
- 状态管理：Pinia
- 样式：原生CSS + CSS变量
- HTTP客户端：预留axios接口

### 项目结构
```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
├── views/           # 页面组件
│   ├── Home.vue     # 首页
│   ├── Translate.vue # AI翻译
│   ├── Shop.vue     # 购物平台
│   └── Stories.vue  # 民俗故事
├── router/          # 路由配置
├── stores/          # 状态管理
└── App.vue          # 根组件
```
